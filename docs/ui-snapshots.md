# UI Snapshots Checklist

- [ ] Idle: `Ready` visible, progress bar visible at `0%`, no global scroll.
- [ ] 1 track + folder ON: layout unchanged, release name enabled.
- [ ] 1 track + folder OFF: release name disabled/dimmed, no layout jump, no extra helper text.
- [ ] 4 tracks: only tracks list scrolls; export panel width unchanged.
- [ ] Rendering: status line + progress bar always visible inside fixed window.
- [ ] Cancelled: `Cancelled` message visible and progress area remains visible.
